# Sepolia Faucet UI

A modern, premium Web3 UI for a Sepolia Testnet Faucet. Built with React and Vite.
This project was scaffolded with a neon/dark-mode aesthetic and features mocked Twitter task verification and mock Token transfers.

## Getting Started

Because Node was unable to run in the current terminal, the source code was written manually. To run the project locally, please use your standard terminal environment where Node.js is installed.

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## Features
- **Wallet Connection:** Supports MetaMask (`window.ethereum`) with a seamless UI, and fallback mock connection.
- **Task Verification:** Simulates API verification of Twitter tasks (Follow and Retweet) with loaders and success states.
- **Token Request:** Premium UI to simulate transferring tokens on the testnet.

Enjoy exploring the code and the beautiful neon-gradient designs!
